# Methodology

## Objective

To evaluate voltage profiles and transmission losses in a small power system network.

## Procedure

1. Define bus data.
2. Construct Y-Bus matrix.
3. Run Gauss-Seidel load flow.
4. Run Newton-Raphson load flow.
5. Compare convergence performance.
6. Calculate transmission losses.

## Assumptions

- Balanced three-phase system.
- Steady-state operation.
- Constant load model.
